# PP4GraphicsEngine

CAMERA CONTROLS: 
W - forward 
A - left 
S - backward 
D - right 
X - down 
SPACE - up Camera 
Rotation - HOLD down Right Mouse Button

G - toggle the grid on/off 
Q - toggle the quad on/off

ANIMATION CONTROLS
P - Start/Stop animation playback
<(',' comma) - step back a frame
>('.' period) - step forward a frame
R - reset back to frame 0 (T pose)
